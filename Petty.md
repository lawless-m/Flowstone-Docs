# Petty

A TypeScript pivot table library with zero dependencies — runs in Node.js and the browser with HTML, CSV, JSON, and Excel export.

**Repo:** `~/Git/Petty`
**Themes:** [[javascript]]

Petty implements Excel-like pivot tables with configurable rows, columns, and aggregation functions (sum, count, average, min, max). Multiple specialised HTML versions: flexible fields, monthly forecasts, multi-dimensional pivots, and standalone with embedded data. Designed for static hosting on Apache with no backend.
