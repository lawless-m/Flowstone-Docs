# HtmlWidgets

A collection of standalone HTML/[[javascript]] widgets — currently a financial date filter picker with hierarchical drill-down, shared with [[DatePickMe]].

**Repo:** `~/Git/HtmlWidgets`

Minimal project containing reusable browser widgets with no framework dependencies. The main component is a date picker with financial period grouping (halves, quarters, months) and range generation, designed for embedding in dashboards.
