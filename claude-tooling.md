# Claude tooling

Projects in this corpus that are either tools *for* Claude Code (skills, commands, extensions, session plumbing) or systems where Claude is a first-class runtime component rather than just something used during development. This is the largest thematic cluster in the corpus after [[rust]].

## Members

- [[Claude-Skills]] — the template repo holding every Claude Code skill and slash command Matt uses
- [[Robocyril]] — blog backend driven by Claude via the `/blog` slash command, in the Cyril persona
- [[StinkySpy]] — ONI data layer with Claude as the query interface, ships its own skills
- [[Gwen]] — QwenCoder Bot, autonomous Rust coding agent that escalates to Claude Code
- [[TheHand]] — voice-to-Claude-Code prompt pipeline for dictating commands
- [[Cilq]] — browser bridge letting Claude drive live web pages and capture console state
- [[Devolver]] — the `devlog` tool that preserves Claude Code session transcripts across context compression
- [[GitIssueLooper]] — pristine template clone; planned Git-issue-looping Claude agent
- [[2claudes1cup]] — MCP bridge letting two Claude instances share state
