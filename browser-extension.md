# Browser extension

Browser-side quality-of-life hacks in this corpus — extensions (usually Chrome / Edge / Vivaldi, Manifest V3) that patch some aspect of the web experience, often with a small native helper behind them.

## Members

- [[DLPlug]] — Chrome/Edge extension plus native messaging host that streams web videos to VLC via yt-dlp
- [[ShushYT]] — Vivaldi extension that auto-mutes YouTube tabs until the real video starts, killing pre-roll ads
