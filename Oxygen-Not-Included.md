# Oxygen-Not-Included

Klei's colony simulation game, and the two corpus projects working with its data and runtime. Together they form a complete "decompile the game → extract building data → build mods against the same assemblies" workflow.

## Members

- [[ONI-Mods]] — C# gameplay mods built against Unity assemblies (storage tooltips, magic storage, and several WIP mods)
- [[StinkySpy]] — Rust production-chain calculator built on data extracted directly from the decompiled game source, not wiki approximations
