# PgQuery

A [[CSharp]] CLI tool for executing SQL against [[PostgreSQL]] with JSON-based connection config.

**Repo:** `~/Git/PgQuery`

PgQuery takes a JSON config file (host, database, credentials), accepts SQL inline or from a file, and outputs formatted tabular results. Supports both SELECT and non-query commands. Deployed to a network share for team access. Hosted on [[Gogs]] at [[Dw.ramsdenInternational.com]].
