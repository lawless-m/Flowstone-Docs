# 9P

Plan 9's filesystem protocol — a simple, synchronous, tagged message protocol for mounting arbitrary trees over the network. Matt has two projects touching 9P from opposite sides.

## Members

- [[Crabbit]] — Rust 9P server exposing a WireGuard tunnel as `/net`, with p9sk1 auth
- [[Nawin]] — .NET 9P2000 protocol library with p9sk1 and dp9ik auth, full client and server
