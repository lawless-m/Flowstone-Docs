# CozoDB

The Datalog-based graph and document database by Ziyang Hu — fast, transactional, and embeddable, with support for multiple storage backends including redb and SQLite. Matt uses it as the backing store for Flowstone's knowledge graph and has a separate repo exploring the redb backend in more detail.

## Members

- [[Flowstone]] — Markdown-to-CozoDB knowledge-graph CLI (the tool that consumes this very corpus)
- [[cozordb]] — Matt's Rust-only fork (`lawless-m/cozo-redb`) with a redb-only storage backend and a working WASM build
