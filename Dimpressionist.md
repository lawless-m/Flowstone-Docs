# Dimpressionist

A self-hosted conversational image generation system using FLUX.1-dev — like ChatGPT's DALL-E but running locally with no content guardrails.

**Repo:** `~/Git/Dimpressionist`
**URL:** `https://dw.ramsden-international.com/dimpressionist/`

[[Python]] web and CLI app that generates images and refines them through natural language ("make the ball red"). Uses FLUX.1-dev on GPU with session persistence, full history, and seed control. Web UI streams progress via WebSocket. Deployed on [[dw.ramsden-international.com]].
