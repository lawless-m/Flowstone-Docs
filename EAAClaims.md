# EAAClaims

A SQL-based data validation tool for checking product, supplier, and shipment data quality in the EAA claims process.

**Repo:** `~/Git/EAAClaims`
**Themes:** [[sage-x3]]

Early-stage project that validates and joins product, ingredient, supplier, and order/quote data from [[sage-x3]], checking for missing Meursing codes, import tariffs, ingredients, packaging, nutrition data, and supplier IDs. Returns per-field missing/OK status for trade compliance review.
