# Ollama

Projects in this corpus that use Ollama for local LLM inference. Matt's home setup runs Qwen 2.5 models on an NVIDIA 3090 via Ollama, with shared VRAM management handed out across several services.

## Members

- [[Caturiel]] — uses Qwen models to decide what to post and write the commentary
- [[Gwen]] — QwenCoder Bot, two-tier agent using 7B for issue triage and 14B for implementation
- [[Ferret]] — small web chatbot using `qwen2.5:7b` by default, with web search tools
